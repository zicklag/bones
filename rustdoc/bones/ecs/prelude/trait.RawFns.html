Expand description

Helper trait that is auto-implemented for all Clone-able types. Provides easy access to drop and clone funcitons for raw pointers.

This simply serves as a convenient way to obtain a drop/clone function implementation for UntypedResource or UntypedComponentStore.

Note: This is an advanced feature that you don’t need if you aren’t working with some sort of scripting or otherwise untyped data access.

Example

let components = unsafe {
    UntypedComponentStore::new(Layout::new::<String>(), String::raw_clone, Some(String::raw_drop));
};

Required Methods

unsafe extern "C" fn raw_drop(ptr: *mut u8)

Drop the value at ptr.

Safety
  • The pointer must point to a valid instance of the type that this implementation is assocated with.
  • The pointer must be writable.

unsafe extern "C" fn raw_clone(src: *const u8, dst: *mut u8)

Clone the value at src, writing the new value to dst.

Safety
  • The src pointer must point to a valid instance of the type that this implementation is assocated with.
  • The destination pointer must be properly aligned and writable.
