Expand description

Struct used to run a system function using the world.

This struct is also used internally by the Dispatcher to create a coherent execution sequence.

Fields

initialize: Box<dyn Fn(&mut World) + Send + 'static, Global>

This will be called once by Dispatcher to initialize the system, allowing it to intialize any resources or components in the world.

Usually only called once, but this is not guaranteed so the implementation should be idempotent.

run: Box<dyn FnMut(&World) + Send + 'static, Global>

This is run every time the system is executed

name: &'static str

A best-effort name for the system, for diagnostic purposes.

Implementations

impl System

pub fn initialize(&self, world: &mut World)

Initializes the resources required to run this system inside of the provided World, if those resources don’t already exist.

This is called automatically if you use a Dispatcher, so in most cases it is not required to call it manually.

This is usually only called once, but this is not guaranteed so the implementation should be idempotent.

pub fn run(&mut self, world: &World) -> Result<(), Error>

Runs the system’s function using the provided World

pub fn name(&self) -> &'static str

Returns the underlying type name of the system.

This is not guranteed to be stable or human-readable, but can be used for diagnostics.
